Este portal foi desenvolvido com o intuito de realizarmos o projeto do 3° bimestre. Neste curso vamos aprender sobre a linguagem de marcação HTML, Folha de estilo "CSS" e uma linguagem de programação JavaScript , apenas para quem se interessar sobre o assunto.

Como funciona o Portal do Aluno?

O portal do aluno é construido por 4 modulos onde serão aplicados os conteúdos e cada modulo terá um conhecimento diferente do outro. É possivel ver os conhecimentos de cada modulo na lista abaixo:

Modulo 1

  • Como a internet funciona
  • Dominio de hospedagem
  • Diferença de HTML, CSS e Java Script
  • Front-end e Full stack
  • Instalando todas as ferramentas (Visual Studio Code e GIMP)
  • Parágrafos e Quebras
  • Símbolos e Emoji
  • Imagens, vídeos e áudios
  • Favicon
  • Hierarquia de Títulos
  • Listas Ordenas e não ordenadas
  • Tipos de Links e Âncoras
  • Estilos de CSS

Modulo 2

  • Psicologia das Cores (Introdução)
  • Harmonia de Cores
  • Backgrounds
  • Fontes
  • Alinhamento de textos
  • Id e Class
  • Pseudo-elementos - CSS
  • Modelos de Caixas
  • Sombra nas Caixas
  • Bordas
  • Organização de estrutura - HTML
  • Vídeo Responsivo

Modulo 3

  • Git e Github
  • Hospedagem de sites
  • Manutenção em sites hospedados
  • Clonagem de repositórios
  • Imagens de fundo
  • Posição de imagens no fundo
  • Tabelas e Efeitos
  • Mesclagem de células
  • Alinhamento do conteúdo em tabelas

Modulo 4

  • Iframes
  • Botões de redes sociais
  • Formulários
  • GET e POST
  • Checkbox e Rádio button
  • Menus Responsivos
  • Múltiplas Media Features
  • Midía Queries
